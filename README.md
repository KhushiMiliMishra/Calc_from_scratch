# Java Servlet Calculator

A web-based calculator built using Java Servlets and Apache Tomcat.

## Features

- Supports + - * / operations
- Handles decimal numbers
- Operator precedence support
- Division by zero detection
- Input validation

## Tech Stack

- Java
- Java Servlet API
- Apache Tomcat
- HTML / CSS

## Architecture

Browser → Tomcat Server → Servlet → Calculator Engine → Response

## How to Run

1. Install Apache Tomcat
2. Compile Java files using servlet-api.jar
3. Deploy the project inside Tomcat webapps
4. Start Tomcat server
5. Open:

http://localhost:8080/CalculatorApp
